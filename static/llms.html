  • undesirables-desktop: The official AI Desktop CRM for The Undesirables — a zero-trust, local-first Tauri application that transforms NFT souls into autonomous AI agents. Features include a Market Oracle with 6 trading strategies, voice synthesis, 3D hologram rendering, and local LLM execution.
  • undesirables-mcp-server: Turn any Undesirable NFT into an MCP-compatible AI agent. Exposes 35 autonomous tools (including TCG card grading, eBay depth analysis, 3D rendering, and voice synthesis) to standard LLM clients (like Claude Desktop or Cursor).
  • elizaos-tcg-oracle-plugin: An ElizaOS plugin that gives AI agents direct access to the TCG Oracle API (370K+ trading card products across 25 games, real-time pricing, Monte Carlo simulations, and AI vision grading).
  • plugin-undesirables: The official ElizaOS client Integration plugin. Seamlessly connects our 4,444 autonomous NFT souls to the broader decentralized AI ecosystem, mapping core logic, video production skills, and on-chain behaviors.
  • Registry: The decentralized package directory and official ElizaOS plugin registry for Meme Merchants. A core index allowing local agents to dynamically resolve, download, and load custom software plugins seamlessly.
  • undesirables-x402-server: This very server. A Coinbase x402 micropayment-gated API proxy for The Undesirables TCG Oracle and Shroomy Arbitrage engines.
